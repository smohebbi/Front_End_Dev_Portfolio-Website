  • jamming ⌄

    Deployed website: https://vermillion-zabaione-3da1a2.netlify.app/

    Repository: https://github.com/smohebbi/jamming

    Tech Stack:

    • React.js
    • npm
    • Stateful Component
    • API (spotify)
    • User Authorization using JSON web tokens
    • JavaScript
    • CSS
    • HTML

    APP's functions:

    • Obtain a Spotify Access Token
    • Searching songs in Spotify by using spotify’s API
    • Showing the search results
    • Add Tracks to a Playlist
    • Change the Name of a Playlist
    • Removing a Tracks from a Playlist
    • Making a track list based on the chosen tracks
    • Saves the Playlist to a User's Account

    Description: it is a page, which I developed based on react components for searching the songs of an artist and save them as a playlist in spotify account. I used the Spotify’s API search the songs and saving them in the account. (Note: in order to use the search and save feature it is needed to log in an Spotify account.)
